# 機能要件定義書 (REQUIREMENTS.md)

## プロジェクト概要
**最短でバズを生み出す！Instagramコンテンツ制作マニュアル**

Instagram向けコンテンツ制作を支援するAI搭載Webアプリケーション

---

## FR-001: バズリール検索・収集機能

### 概要
キーワード検索でInstagramのバズリールを検索・収集する機能

### 要件
| 項目 | 内容 |
|------|------|
| キーワード検索 | 任意のキーワードでリール検索 |
| 最小再生数フィルター | 3万回再生以上のコンテンツ抽出 |
| 期間フィルター | 投稿日による絞り込み (6ヶ月以内推奨) |
| ソート機能 | 再生数降順でソート |

### 実装状態: **実装済み (モックデータ)**

---

## FR-002: 動画ダウンロード機能

### 概要
InstagramリールURLから動画をダウンロードする機能

### 要件
| 項目 | 内容 |
|------|------|
| URL検証 | InstagramリールURL形式の検証 |
| ID抽出 | URLからリールID抽出 |
| ダウンロード案内 | 外部サービス (snapinsta.to) 案内 |

### 実装状態: **実装済み (外部サービス案内)**

---

## FR-003: AI台本生成機能

### 概要
動画の説明文からAIで完璧な台本を生成する機能

### 要件
| 項目 | 内容 |
|------|------|
| 入力 | 動画の説明文・概要 |
| 出力 | 完璧な再現用台本 |
| AI | Claude API使用 |

### 実装状態: **実装済み**

---

## FR-004: バズ分析機能

### 概要
台本を分析してバズの理由・パターンを抽出する機能

### 要件
| 項目 | 内容 |
|------|------|
| バズ理由分析 | なぜバズったかの理由抽出 |
| パターン分析 | 共通パターンの特定 |
| 成功要因 | 成功に寄与した要素 |
| 推奨事項 | 改善・活用の提案 |

### 実装状態: **実装済み**

---

## FR-005: Threads投稿生成機能

### 概要
台本からThreads向け2段階投稿を生成する機能

### 要件
| 項目 | 内容 |
|------|------|
| Part 1生成 | 導入・前半部分 |
| Part 2生成 | 後半・結論部分 |
| コピー機能 | ワンクリックコピー |
| 絵文字・太字禁止 | プレーンテキスト出力 |

### 実装状態: **実装済み (API接続完了)**

---

### 概要
既存台本から視点を変えた新しいリール台本を生成

### 要件
| 項目 | 内容 |
|------|------|
| 視点変換 | 異なるアングルからの台本生成 |

### 実装状態: **実装済み**

### 概要

|------|------|
| 入力 | 台本テキスト |

### 実装状態: **実装済み**

---

## FR-008: コミュニケーション加速機能

### 概要
コメント返信・エンゲージメント用コメントアイデア生成

### 要件
| 項目 | 内容 |
|------|------|
| 返信モード | 受信コメントへの返信案生成 |
| エンゲージメントモード | 他投稿へのコメント案生成 |
| 複数案生成 | 3〜5個のアイデア提示 |
| コピー機能 | ワンクリックコピー |

### 実装状態: **実装済み (API接続完了)**
